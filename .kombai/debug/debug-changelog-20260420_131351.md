## Changes Made

### 1. Fixed undefined method call in migration
- **File:** `migrations/m260420_000007_add_dashboard_performance_indexes.php`
- **Change:** Replaced `$index->getColumnNames()` with `$index->columnNames`.
- **Reason:** `yii\db\IndexConstraint` does not have a `getColumnNames()` method; it uses a public property `$columnNames`.
- **Revert:** Change `$index->columnNames` back to `$index->getColumnNames()`.

### 2. Improved error handling in BackupSyncCommand
- **File:** `components/BackupSyncCommand.php`
- **Change:** Moved `$this->resolveBackupConnectionForCurrentDatabase($backup)` inside the `try-catch` block.
- **Reason:** Opening the backup connection was failing with "Unknown database" errors (due to mismatched local/remote db names), which blocked the primary database operations (like migrations).
- **Revert:** Move the `resolveBackupConnectionForCurrentDatabase` call back outside the `try-catch` block.
