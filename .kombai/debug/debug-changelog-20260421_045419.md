## Changes Made

### 1. Fixed "Unknown Property: app\models\User::email"
- **File:** `controllers/FormController.php`
- **Change:** Added a check `if ($identity->hasProperty('email'))` before trying to access the email property of the logged-in user.
- **Reason:** The `users` table and the `User` model do not have an `email` column, but the code was trying to inject it into form submission data. This was the root cause of the Internal Server Error 500.
- **Revert:** Remove the `hasProperty` check and access `->email` directly.

- **File:** `web/index.php`
