## ✨ **QUICK START GUIDE**

### Adding Blocks
1. **Drag from sidebar** - Click and drag any block to the canvas
2. **Click to add** - Single-click blocks in the left panel to add them

### Managing Blocks
- **Move** - Use ↑↓ buttons to reorder
- **Duplicate** - Use ⧉ button to copy
- **Delete** - Use ✕ button to remove
- **Edit** - Click block to select and use Properties panel

### Organizing
- **Search** - Use 🔍 search box to find blocks
- **Categories** - Expand/collapse categories to browse
- **Responsive** - Test on 🖥️ Desktop/📱 Tablet/📲 Mobile view
