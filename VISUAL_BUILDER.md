## 🚀 How to Use

### 1. Create Page
```
Forms → Create New Form
↓
Page opens with empty canvas
↓
Click blocks from left sidebar or drag to canvas
↓
Edit properties in right panel
↓
Click "Publish" to save
```

### 2. Add Blocks
```
Click block in sidebar → Block added to canvas
OR
Drag block from sidebar → Drop on canvas
```

### 3. Edit Block
```
Click block on canvas → Right panel shows properties
↓
Edit Content/Style/Advanced tabs
↓
Changes appear in real-time
```

### 4. Reorder Blocks
```
Drag ⠿ handle up/down
OR
Click ↑↓ buttons on block header
```

### 5. Duplicate Block
```
Click ⧉ button on block header
→ Copy appears below original
```

### 6. Delete Block
```
Click ✕ button on block header
→ Confirm deletion
```

### 7. Preview Page
```
Click "👁️ Preview" in toolbar
→ Opens form in new tab
```

### 8. Save/Publish
```
Click "💾 Publish" in toolbar
→ Form saved to database
```

## 💾 Data Structure

Each block is saved as JSON:

```json
{
  "type": "product-card",
  "label": "Product",
  "name": "Amazing Product",
  "price": "99.99",
  "image": "https://...",
  "badge": "sale"
}
```

Full form schema:
```json
[
  { "type": "hero", "label": "Hero", "title": "Welcome", ... },
  { "type": "product-grid", "label": "Products", "products": 6, ... },
  { "type": "testimonial", "label": "Review", "text": "...", ... },
  { "type": "newsletter", "label": "Subscribe", ... }
]
```
